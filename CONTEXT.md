# Aawej Pathan Portfolio

A professional, high-end minimalist tech portfolio for Aawej Pathan, a 16-year-old AI Engineer and Entrepreneur, showcasing his projects, startup initiatives, coding accomplishments, and interactive AI showcases.

## Language

**Portfolio Site**:
The unified web application displaying the professional profile, projects, and contacts of Aawej Pathan.
_Avoid_: Personal website, home page, CV site

**Project Showcase**:
A curated grid displaying completed and upcoming AI products, specifying tech stacks, roles, and repository links.
_Avoid_: Grid, list, gallery

**Builder Lab**:
A log of rapid prototypes, AI experiments, and product validations, showing active, paused, or archived states.
_Avoid_: Playground, demo widget, project sandbox

**Journey Timeline**:
A chronological record of milestones, accomplishments, coding achievements, and startup progression.
_Avoid_: Resume, history, achievements log

**Social Hub**:
The primary navigation/contact actions linking to X, GitHub, Instagram, and email.
_Avoid_: Footer links, contact links

**Blog Post**:
A static markdown file stored securely in the GitHub repository containing a post's title, date, and content, authored by Aawej Pathan via the GitHub web editor.
_Avoid_: Database record, CMS entry, article file

## Example Dialogue

**Developer**: "We need to structure the Project Showcase. Should upcoming startups and hackathon projects be mixed?"
**Domain Expert**: "Yes, but let's label upcoming startups as 'Launching Soon' to signal entrepreneurship, and link hackathon projects directly to GitHub within the Project Showcase."
**Developer**: "Got it. And for the Social Hub, do we need a complex contact form?"
**Domain Expert**: "No, a minimalist copy-to-clipboard email action and direct links to X, GitHub, and Instagram are cleaner and fit the Minimalist Tech aesthetic better."
